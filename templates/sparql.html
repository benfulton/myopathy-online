Sample Queries
To find the drugs that treat myositis 
prefix rdfs: <http://www.w3/org/2000/01/rdf-schema#>
prefix onto: <http://myopathy-online.herokuapp.com/ontology#>
SELECT ?drug
  WHERE
{
  ?drug onto:treat onto:myositis
}

To find the genes whose mutation may result in neuropathy 
prefix rdfs: <http://www.w3/org/2000/01/rdf-schema#>
prefix onto: <http://myopathy-online.herokuapp.com/ontology#>
SELECT ?gene
  WHERE
{
  ?gene onto:resultIn onto:neuropathy
}

To find the diseases that may be caused by the mutation of gene ACTA1 
prefix rdfs: <http://www.w3/org/2000/01/rdf-schema#>
prefix onto: <http://myopathy-online.herokuapp.com/ontology#>
SELECT ?disease
  WHERE
{
  ?disease onto:becauseOf onto:ACTA1
}
